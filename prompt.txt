
С середины восемнадцатого века и по сей день данные о сотрудниках компании 
«Артель Свѣтлое будущее» традиционно хранят в амбарных книгах. 
Каждая страница книги поделена на два столбца: в левом указаны имя и фамилия 
сотрудника, а в правом — дата рождения:
Акакій Башмачкинъ	23 марта 1791 года
Яков Степанов	Двадцать шестое июля 1971
Потап Алексеев	16.09.1990
Евгений Женин	5 декабря 1984
Кондрат Александров	18.01.1994
Прогресс не стоит на месте, и в XXI веке список всё же пришлось оцифровать. 
При оцифровке предполагалось, что даты будут в формате ДД.ММ.ГГГГ, однако при 
заполнении книг не все писари, клерки и менеджеры соблюдали этот формат и 
писали даты как попало.
Напишите программу, которая получит на вход такой список кортежей с данными:

data = [
    ('Акакій Башмачкинъ',    '23 марта 1791 года'),
    ('Яков Степанов', 'Двадцать шестое июля 1971'),
    ('Потап Алексеев', '16.09.1990'),
    ('Евгений Женин', '5 декабря 1984'),
    ('Кондрат Александров', '18.01.1994')
] 
Программа должна подсчитать: 
количество записей, где дата указана корректно;
количество записей, где дата указана некорректно.
В ходе работы программа должна вывести такой результат:

Некорректный формат даты в записи: Акакій Башмачкинъ, 23 марта 1791 года
Некорректный формат даты в записи: Яков Степанов, Двадцать шестое июля 1971
Некорректный формат даты в записи: Евгений Женин, 5 декабря 1984
Корректных записей: 2
Некорректных записей: 3 
В программе должны быть две функции:
Функция process_people() должна подсчитать количество корректных и некорректных
 записей.

Эта функция принимает на вход список кортежей entries. В каждом из кортежей 
первый элемент — имя, а второй — дата рождения.

В функции должны быть объявлены два счетчика:

good_count для подсчёта корректных записей;
bad_count для подсчёта некорректных записей.
В каждом кортеже из списка entries надо проверить, в правильном ли формате 
указана дата рождения сотрудника. Для этой проверки должна вызываться функция
 validate_record(). В результате проверки должно увеличиться значение одного
  из счётчиков. 
В конце работы функция должна вернуть словарь с ключами good и bad, где 
значениями ключей будут значения счётчиков good_count и bad_count 
соответственно.
Функция validate_record() принимает на вход два параметра: имя сотрудника 
name и дату его рождения birthdate. 
С помощью метода datetime.strptime() функция должна проверить, соответствует
ли полученная строка с датой рождения формату ДД.ММ.ГГГГ. 
Если строка не соответствует ожидаемому формату — метод datetime.strptime()
выбросит исключение. 
Обработайте это исключение:
Напечатайте сообщение 

Некорректный формат даты в записи: <имя>, <дата_рождения> 
Верните из функции False.
Если же дата рождения соответствует ожидаемому формату — верните True.
