
Для этого напишите функцию universal_converter(), которая принимает:
code — строка, обозначающая двухбуквенный код страны пребывания;
rub_amount — целое число, обозначающее количество рублей, которое нужно 
перевести в местную валюту. По умолчанию — 0.
И возвращает строку:
«Местная дата и время: <текущие дата и время в местном формате>, 
<rub_amount > RUB = <количество местной валюты, эквивалентное количеству 
рублей с точностью 2 знака после запятой> <валютный тикер в верхнем регистре>».
Также функция должна выполнить проверку: корректно ли введены параметры:
rub_amount не может быть меньше нуля,
code может быть написан в любом регистре (или верхним, или нижним, 
но не их пересечение). Код принимается только стран, указанных в таблице. 
В случае обнаружения ошибки программа возвращает строку «Ошибка ввода: 
<имя параметра который введён неверно> <его значение>».
Сначала проверьте rub_amount, затем code. Все валютные вычисления должны 
происходить с использованием Decimal.

Напишите вспомогательную функцию conversion_result, которая принимает:
dt_format — строка, обозначающая формат времени;
rub_amount — целое число, обозначающее количество рублей, которое нужно 
перевести в местную валюту. По умолчанию — 0;
currency_rate — вещественное число, обозначающее курс валюты;
currency_symbol — строка, обозначающая тикер (трёхбуквенный код) валюты.
И возвращает результирующую строку:
Страна	Код страны	Валютный тикер	Курс валюты к рублю	Формат времени
Германия	DE	EUR	93.763	25.12.2023 14:30 Uhr
Китай	CN	CNY	11.164	2023-12-25 14:30
США	US	USD	79.322
