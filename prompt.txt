Торговая сеть ООП заботится о своих посетителях и предоставляет им скидки.
В программе, которая управляет ценами на товары, надо реализовать механизм 
вычисления цены с учётом скидки конкретного клиента. 
В прекоде объявлен класс Customer, в нём есть только один атрибут — name, 
он хранит имя пользователя.
Допишите класс Customer.
В инициализаторе класса Customer создайте приватный атрибут, который будет 
хранить значение скидки для конкретного клиента (в процентах). По-английски 
«скидка» — это discount, так этот атрибут и назовите. У этого атрибута должно 
быть значение по умолчанию: 10.

Этот атрибут не должен быть доступен для изменений извне.
В этом же классе реализуйте метод get_price(). Этот метод должен
принять на вход исходную цену товара,
вычислить новую цену: к полученной цене применить скидку и округлить результат 
до двух знаков после запятой,
вернуть новую цену.
Реализуйте метод set_discount(), который принимает на вход новое значение 
скидки и заменяет значение соответствующего атрибута.

Магазин не может предоставить скидку более 80%. В методе set_discount() 
реализуйте проверку: если этот метод получит на вход значение более 80 — 
должно быть сохранено значение 80.

Имя приватного атрибута класса должно начинаться с двух нижних подчёркиваний, 
например, __attribute.
Чтобы к цене применить, например, скидку в 30%, необходимо умножить исходную 
цену на 0.7.
Для округления примените функцию round().
