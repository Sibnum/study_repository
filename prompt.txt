На каждой итерации цикла для каждой буквы исходной фразы найдите её индекс в 
алфавите alphabet.
Чтобы получить индекс нужного символа в строке alphabet — примените метод 
alphabet.find(<символ>).
При поиске символов примените метод <символ>.lower() — иначе не найдутся индексы
 букв, написанных в верхнем регистре.
Чтобы при шифровке или дешифровке новый индекс остался в границах длины 
алфавита — получите нужный индекс через вычисление остатка при делении на 33 (при помощи оператора %).

ТЗ:
Требования к классу CipherMaster:
Текст, переданный для шифрования, может содержать буквы в любом регистре.
Метод cipher() должен принимать исходную строку и целое число, означающее сдвиг
по алфавиту. Этот метод должен возвращать зашифрованную строку.
Метод decipher() должен принимать зашифрованную строку и целое число, означающее
 сдвиг по алфавиту. Этот метод должен возвращать расшифрованную строку.
Текст, возвращаемый методами cipher() и decipher(), должен быть приведён к 
нижнему регистру (не должен содержать больших букв).
Любые символы, не входящие в строку alphabet, не должны заменяться.
