Создайте генераторное выражение, которое из списка чисел numbers возводит в 
квадрат те, что делятся на 3. Затем распечатайте сумму квадратов.
Примеры: 
Для списка numbers = [1, 3, 4, 6, 9, 11] ваша программа должна вывести сумму 
квадратов чисел 3, 6 и 9, то есть 3**2 + 6**2 + 9**2 = 126.
Для списка numbers = [1, 4, 11] программа должна вывести 0.

Для решения этой задачи необходимо написать генераторное выражение, 
а не функцию-генератор — то есть в решении не нужно объявлять функцию и не 
нужно применять инструкцию yield.
Чтобы отфильтровать числа, которые делятся на 3, используйте операцию получения 
остатка при делении. Для этого вам понадобится оператор %.
Используйте оператор ** для возведения числа в квадрат.
Используйте функцию sum() для подсчёта суммы элементов, полученных из 
генераторного выражения.
